Features

Everything you need for a fast, credible security assessment.

🏗

Single Executable

Drop winposture.exe on a USB drive and run it on any Windows machine. No installation, no Python, no dependencies.

🔒

Fully Offline

No data leaves the machine. No telemetry, no cloud APIs, no license servers — your audit data stays yours.

📋

CIS Benchmark Mapping

Every finding is annotated with the corresponding CIS Benchmark control ID — v5.0.0 for Windows 11, v4.0.0 for Windows 10 — detected automatically at scan time.

📈

Scored & Graded

A 0–100 security score with letter grade gives you an at-a-glance picture of the system's risk posture.

📄

HTML & JSON Reports

Export a self-contained HTML report for stakeholders or a structured JSON report for pipeline integration.

🔄

Comparative Scanning

Save a baseline and diff against it later to track remediation progress over time with --baseline and --compare.

⚙

Custom Profiles

A winposture.toml config lets you disable checks, override severities, and tune thresholds for your environment.

🔧

Actionable Remediation

Every failing check includes a concrete fix command or Group Policy path — not just a problem description.

Get Started

Up and running in under a minute.

1

Download the executable

Grab winposture.exe from the latest release and save it somewhere easy to find — your Desktop or Downloads folder works fine.

2

Open an Administrator terminal

Right-click Command Prompt or PowerShell → Run as administrator. Some checks (BitLocker, local accounts, services) require elevation for full results.

3

Navigate to the folder where you saved the file

For example, if you saved it in Downloads:

cd %USERPROFILE%\Downloads

Tip: in File Explorer, navigate to the folder, then type cmd in the address bar and press Enter to open a terminal there directly.

4

Run a scan

winposture.exe
5

Generate an HTML report (optional)

winposture.exe --html report.html --verbose

Opens as a standalone file in any browser — no server required.

6

Track changes over time

winposture.exe --baseline before.json
# ... remediate findings ...
winposture.exe --compare before.json
